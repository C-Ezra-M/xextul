Xextul (SHEHSH-tool) is a Xextan toolkit made by C.Ezra.M. However, right now it only contains one tool, which is a table of root words.

Sitste zakvla / Table of root words

This can be used to find root words that have yet to be filled. Gray cells denote unassigned roots.
